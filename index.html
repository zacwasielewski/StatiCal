    Welcome to StatiCal

    StatiCal is a very simple static calendar widget for WordPress. It allows the site author to specify a title, year, and month calendar to display. Optionally, one or more days may be highlighted. That's it!

    Installation

    1. Upload the statical/ directory to the /wp-content/plugins/ directory
    2. Activate the plugin through the 'Plugins' menu in WordPress
    3. Drag the widget to your desired location in the 'Appearance' / 'Widgets' menu in WordPress
